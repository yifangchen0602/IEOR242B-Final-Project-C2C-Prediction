Final Best Comparison Result
============================

Evaluation set:
  original_test, using the original non-SMOTE validation/test distribution.

Best overall model:
  Multi-seed Contrastive Ensemble

Method:
  - Train the final projection-head corruption Contrastive + MLP model with
    five seeds: 42, 7, 13, 21, and 84.
  - Average the five predicted probabilities.
  - Select the final threshold on validation by requiring precision >= 0.45
    and then maximizing recall.
  - Final threshold = 0.552832.

Best final result on original_test:
  accuracy  = 0.9356
  precision = 0.4534
  recall    = 0.5127
  F1        = 0.4813
  ROC-AUC   = 0.8701
  PR-AUC    = 0.4583

Best single neural model:
  Projection-head Corruption Contrastive + MLP

Single-model result on original_test:
  accuracy  = 0.9329
  precision = 0.4365
  recall    = 0.5208
  F1        = 0.4749
  ROC-AUC   = 0.8680
  PR-AUC    = 0.4526

Best baseline by F1:
  MLP
  F1     = 0.4379
  PR-AUC = 0.4195

Best baseline by PR-AUC:
  Random Forest
  F1     = 0.4331
  PR-AUC = 0.4569

Main conclusion:
  The raw multi-seed contrastive ensemble is the strongest final model. It
  improves F1 over the best baseline MLP from 0.4379 to 0.4813, improves F1
  over the best single contrastive model from 0.4749 to 0.4813, and slightly
  improves PR-AUC over Random Forest from 0.4569 to 0.4583.
